Если параметры не внесены пользователем, то по умолчанию будут использоваться 5 сек и 3 мин.

1. Частота опроса программы (секунды).
     Описание: информация о процессах фиксируется в текстовый документ по установленному времени. Интервал времени перед аварийным закрытием программы не вносится. 
     time_check_sec = 5

2. Время зачёта простоя (минуты).
     Описание: после установленного времени учёт активности программ временно прерывается.
     time_afk = 3

3. В разработке. Установлена сортировка по разделам по умолчанию.
  
     Sections = 
     	Steam: steam,
    	Браузер: google, chrome, yandex,
     	Мессенджеры: discord, telegram,
    	Работа: code
